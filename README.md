A personal learning journal to log what you learn every day, reflect on missed days, and visualize your consistency over time with heatmaps.
## Features
1. Daily reflection - What did I learn today?
2. Accountability - Which days did I miss, and why?
3. Long-term insight - How consistent am I over weeks and months?
4. Daily Learning Log - Prompt appears automatically each day, can add multiple entries a day
5. Manual Tags per Entry - Add optional tags, useful for topic tracking and later analysis
6. One-click backup of the local SQLite database
7. Runs locally using Streamlit, Launchable via desktop shortcut

## Monthly Analytics Dashboard
1. Days logged vs missed
2. Total entries
3. Average entries per active day
4. Editable reasons for missed days
5. GitHub-Style Heatmap to visualize consistency

### Tech Stack - Python, Streamlit, SQLite
